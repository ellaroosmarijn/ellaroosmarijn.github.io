JavaScript in Relation to HTML and CSS.

What is Javascript?

Javascript is a programming language that is used to create interactive effects on the web and to build web-based applications. In short, Javascript adds behaviour to websites.

The Relationship Between Javascript, HTML and CSS.

HTML is the ‘bones’ or structure of a webpage. On its own it prepares the content of a webpage with no formatting. CSS then formats and styles the HTML content, to make a more user friendly interface. Javascript is a third and very important layer of a webpage. Javascript allows the webpage to be interactive with the user by returning a value or change according to the set action. Without Javascript a webpage created solely from HTML and CSS may be less engaging with the user due to a less responsive design.

What is an Analogy for the Relationship Between Javascript, HTML and CSS?

An analogy for this relationship would be that of the human body. The HTML would be the skeleton, since it forms the bones of the webpage. CSS would be the skin because it provides styling and formatting that make the webpage more appealing to a user, just as skin can change how someone looks. Javascript would be the muscles because it adds behaviour and interactive features to a website, like muscles provide movement and behaviour in people.

Control Flow and Loops.

What is Control Flow?

Control flow in programming is the order in which statements and functions must be made. This is because a function cannot be called before it has been defined, just as you would name a puppy before trying to call it over to you. Control flow allows for the developer to choose which sections of code to run and when. An example of control flow in coding would be:

function eatPie(pie) { if (pie <= 10) { text = “Eat pies!”; } else { text = “Too many pies to eat!”; } }

Here, the function eatPie has been defined, which means that if we want to ‘call’ the function (have it shown on the webpage) we can call it after this function. The next part of the code is the if-else statement. The if else statement is telling the computer which code to run and when. So if there are ten or more pies, the webpage will show the text “Eat pies!”. The ‘else’ part of the if-else statement tells the computer that only if there are more than ten pies to show the text “Too many pies to eat!”.

What is a Loop?

A loop is typically written in Javascript and is a sequence of code or instructions that are repeated until a condition has been met. Generally a loop runs a specific number of times, or until it has found or changed specific data that was predefined. An everyday example of a loop is when you shop with a grocery list. A grocery list could be considered as an array (a collection of items) that the shopper must read through to check off every item. The action of the shopper checking the list repeatedly is like a loop that will only cease to occur when there are no items left on the list. An example of a loop would be written as:

function groceryList(‘apples’, ‘bread, ‘milk’) { for (let i = 0; i < 3; i++) { console.log(groceryList[i]) } }

Here, the funcion groceryList has an array (list) of food items as a parameter. It will begin at the first item (as indicated by i = 0) and continue through each item in numerical order (as indicated by i++) until it reaches the third and last item (as indicated by i < 3). This is essentially the written version of what someone would do when they go through their grocery list at the supermarket.

Since repetitive tasks are reasonably common in programming, loops become important for minimising the chance of bugs and saving time by only writing a task once.

What are Functions and Why Are They Helpful?

What is a Function?

Essentially every programming language lets you create ‘blocks’ of code that perform specific tasks. The blocks of code are called functions. They are helpful because they enable a developer to write a function once and recall it in multiple places on the file. Additionally, it helps with breaking a larger set of instructions or tasks into smaller chunks. A function can be written as:

function goodMorning(name) { return ‘Good morning ’ + name } let goodMorningResult = goodMorning(‘Maye’)

Here, the function is goodMorning that has a parameter of a name. The name is defined at the bottom of the code a ‘Maye’, although any name can be entered here. The function will then return the text “Good morning Maye”.

Arrays and Objects.

What is an Array?

An array is a data structure consisting of a collection of items (e.g. variables or values). In an array each item of data has an index number which starts at zero. For instance, in the image below the array contains the values [73, 98, 86, 61, 96]. In this array 73 has an index number of 0.

You can access the data in an array by referring to its index number i = 3, like in the earlier example of the code for a loop. On the other hand, accessing data from an object is a very different process.

What is an Object?

An object is a key value storage structure with ‘prototypical inheritance’ where if a key is not on an object, it will check if the key is on a parent. An object can inherit methods and values from another object. To access an object you need a variable with a reference that contains the object. From here you can either use the dot notation or the bracket notation. Let’s say that we wanted to find the length of the object:

let names = [‘Maye’, ‘Freya’, ‘Noah’]

With dot notation you would use names.length, and with bracket notation you would use names[length]. In this scenario the code would return a value of three, because there are three values.

Editing in the DOM.

This piece of code Click this link is represented by four nodes in the DOM.

  1. The HTMLAnchorElement, which is the parent element.
  2. The HTMLAnchorElement.href (this is an attribute node), which is a child node.
  3. The HTMLAnchorElement.target (this is also an attribute node), which is a child node.
  4. The text node ‘Click this link’, which is a child node.

To edit the anchor element in the DOM, you will first need to go to the document interface. Once you’re in the interface you can use either the document.querySelector() to find the first selector or the document.querySelectorAll() to find all of the selector. In this scenario, we’ll use the document.querySelector() since we have written only one anchor element. The code should look like this:

let element = document.querySelector(‘a’);

We have now selected the first ‘a’ element. Let’s say that we want to change the colour of the link from blue to green. To do this, we will use the HTMLElement.style property to change the CSS on the HTML anchor tag. This specific tag is used because the anchor tag is a child of the HTMLElement node, and will inherit its properties. The .style.color is added to the HTMLElement to reference the CSS (style) and that we want the colour to change (.color). The code for changing the colour should look like this:

element.style.color = ‘green’;

If we had more than one anchor element we could have used the document.querySelectorAll() to change all of the anchor element’s colours from blue to green.
